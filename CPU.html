Prueba de Ensamblaje y Encendido

Descripción de la actividad: Desmontaje y reensamblaje de un equipo de cómputo

Durante esta práctica, se realizó el desmontaje completo de una computadora con el objetivo de identificar cada uno de sus componentes internos. Posteriormente, se procedió a volver a ensamblar todos los elementos para confirmar el funcionamiento correcto del sistema.

El proceso comenzó apagando completamente el equipo y desconectándolo de la corriente eléctrica para garantizar la seguridad. Luego, se retiraron los paneles del gabinete para tener acceso al interior del sistema.

A continuación, se desmontaron de forma ordenada los componentes principales: la fuente de alimentación, la tarjeta madre, los módulos de RAM, el dispositivo de almacenamiento, la tarjeta gráfica (si estaba presente), el disipador del procesador, así como los cables de energía y datos.

Durante esta etapa, se observó la ubicación y función de cada componente, reforzando el entendimiento sobre la estructura interna de una computadora.

El proceso de reensamblaje siguió este orden:

  • Fuente de alimentación
  • Placa base
  • Procesador y su sistema de refrigeración
  • Memoria RAM
  • Tarjeta gráfica
  • Unidad de almacenamiento
  • Conexiones de energía y datos

Finalmente, se cerró el gabinete y se conectó nuevamente el equipo para realizar una prueba de encendido. El arranque exitoso del sistema indicó que el ensamblaje fue realizado correctamente.

Esta actividad permitió a los participantes adquirir experiencia práctica en el desensamblado y armado de equipos, comprendiendo la relevancia de una correcta instalación para garantizar el buen funcionamiento de la computadora.
